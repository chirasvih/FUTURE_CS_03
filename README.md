# Secure File Sharing System

## Project Description
This project is a secure file sharing system built using Python Flask.
Files uploaded by users are encrypted using AES encryption before being stored.
During download, files are decrypted securely and sent to the user.

## Technologies Used
- Python Flask
- AES Encryption (PyCryptodome)
- HTML
- Git & GitHub

## Key Features
- Secure file upload
- AES encryption at rest
- Secure decryption during download
- Simple and user-friendly interface

## How to Run
1. Install dependencies.
2. Run the app.
3. Open browser:
   http://127.0.0.1:5000
